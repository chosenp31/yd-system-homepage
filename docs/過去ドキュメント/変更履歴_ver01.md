# 変更履歴 ver01

## 概要
ydシステム会社コーポレートサイトの変更履歴

---

## ver01 (2026-01-15)

### 初回リリース

#### 新規作成
| 対象 | 内容 |
|------|------|
| プロジェクト | Next.js 16 + TypeScript + Tailwind CSSでプロジェクト作成 |
| トップページ | ヒーロー、お悩み、対応業界、スクラッチ開発の強み、選ばれる理由、開発実績、お客様の声、FAQ、CTAセクションを実装 |
| サービスページ | スクラッチ開発サービスの詳細、パッケージとの比較表を実装 |
| 実績ページ | 5つの開発実績の一覧・詳細ページを実装 |
| 会社概要ページ | 会社情報を実装 |
| お問い合わせページ | Formspree連携の問い合わせフォームを実装 |
| ヘッダー | ロゴ、ナビゲーション、モバイルメニューを実装 |
| フッター | 会社情報、ナビゲーション、コピーライトを実装 |
| デプロイ | Vercelへ本番デプロイ完了 |

#### LP改善（同日実施）
| 対象 | 変更前 | 変更後 |
|------|--------|--------|
| ヒーローセクション | 実績数字なし | 50+件、98%継続率、10年以上の実績を追加 |
| CTAボタン | 「お問い合わせ」 | 「無料で相談する」に変更 |
| 選ばれる理由セクション | なし | 業界特化の知見、一気通貫サポート、柔軟な対応の3項目を追加 |
| お客様の声セクション | なし | 3件のテスティモニアル（不動産、人材、販売代理店）を追加 |
| FAQセクション | なし | 5つのFAQ項目をアコーディオン形式で追加 |
| 実績カード | 成果なし | 成果バッジ（「追客漏れ90%削減」等）を追加 |

#### 技術的な修正
| 対象 | 内容 |
|------|------|
| メタデータエラー | service/page.tsxの`export const metadata`をlayout.tsxに移動（Client Componentでのmetadataエクスポート不可のため） |
| 各ページlayout.tsx | service, works, company, contactの各ディレクトリにlayout.tsxを作成してメタデータを設定 |
