# 変更履歴 ver03

## 概要
ydシステム会社コーポレートサイトの変更履歴

---

## ver03 (2026-01-16)

### コーポレートサイトへの完全移行

20社以上のシステム開発会社サイトを調査し、LP（ランディングページ）要素を完全に削除。シンプルな会社紹介サイトに再構成。

#### 削除したセクション
| セクション | 理由 |
|-----------|------|
| 「私たちが解決できること」 | LP特有のペインポイント訴求 |
| 「スクラッチ開発の強み」 | 過剰なセールス訴求 |
| 「ydシステムが選ばれる理由」 | LP特有の差別化訴求 |
| 「お客様の声」（テスティモニアル） | LP特有の社会的証明 |
| 「よくあるご質問」（FAQ） | LP特有の不安解消セクション |
| 実績数字（50+件、98%） | LP特有のファクト訴求 |
| グラデーションCTAセクション | 過剰な誘導デザイン |

#### 残したセクション（5つのみ）
| セクション | 内容 |
|-----------|------|
| Hero | シンプルなタイトル + 説明文 + お問い合わせボタン |
| 事業内容 | 3項目（業務システム開発、保守・運用、コンサルティング） |
| 開発実績 | 3件のカード表示 + 一覧へのリンク |
| 会社紹介 | 会社名、所在地、事業内容の基本情報 |
| お問い合わせ | シンプルなCTA |

#### 修正ファイル
- src/app/page.tsx（全面書き換え）

## ver02 (2026-01-16)

### コーポレートサイトらしさの強化

#### 削除・変更
| 対象 | 変更前 | 変更後 | 理由 |
|------|--------|--------|------|
| ヒーローのサブタイトル | 「不動産・人材・販売代理店に特化」 | 「業務システムのスクラッチ開発」 | 特化していないため削除 |
| ヒーローのメインコピー | 「パッケージでは解決できない業務課題を解決する」 | 「お客様の業務に合わせたシステムを開発します」 | 会社紹介らしいトーンに |
| 実績数字 | 「50+件」「98%」「10年以上」の3つ | 「50+件」「98%」の2つ | 10年以上は事実でないため削除 |
| お悩みセクション | 「こんなお悩みはありませんか？」 | 「私たちが解決できること」 | LP感を減らす |
| 対応業界セクション | 不動産・人材・販売代理店の3業界 | セクション削除 | 特化を謳わない |
| 選ばれる理由 | 「業界特化の知見」 | 「丁寧なヒアリング」 | 業界特化を謳わない |
| CTAボタン | 「無料で相談する」 | 「お問い合わせ」 | 会社紹介らしいトーンに |
| テスティモニアル業種 | 「不動産仲介会社」「人材派遣会社」「販売代理店」 | 「サービス業」「卸売業」「製造業」 | 業界特化を謳わない |

#### 修正ファイル
- src/app/page.tsx
